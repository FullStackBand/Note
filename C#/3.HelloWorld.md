## 4.1命名空间

### 1. 概念

   C#程序中的一种代码组织形式,主要用来标识类的可见范围

### 2. 语法

   * 声明

     `namespace 命名空间名称`

   * 引用

     `using 命名空间名称`

   * 例如

     ```C#
     using System;//命名空间
     using mySpace;//使用命名空间
     
     namespace HelloWorld //命名空间
     {
         class Program //类
         {
             Text text = new Text();
             static void Main(string[] args)// Main方法
             {
                 // 输出内容
                 Console.WriteLine("Hello World!");
                 Console.ReadLine();
             }
         }
     }
     namespace mySpace // 命名空间
     {
         class Text // 类
         {
     
         }
     }
     ```
